# VirusSpreader9000

Spreads a chosen role through replies.

## How it works
- Reply to someone with the Virus role
- You get infected

## Setup
Create a `.env` file:

DISCORD_TOKEN=your_token  
VIRUS_ROLE_ID=your_role_id  

Run:
node index.js

## Notes
- Bots, DMs, and self-replies are ignored
- 😷 or :mask: prevents infection
